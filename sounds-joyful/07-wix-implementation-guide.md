# Wix Implementation Guide — Sounds Joyful Workplace Wellbeing

This guide walks you through adding the Workplace Wellbeing section to the existing Sounds Joyful Wix site.

---

## Page Structure

| Page | File | Purpose |
|------|------|---------|
| Workplace Wellbeing | 01-workplace-wellbeing-overview.md | Main landing page, overview of all three experiences |
| The Reset Experience | 02-reset-experience.md | Individual page for the 45–60 min session |
| The Connection Experience | 03-connection-experience.md | Individual page for the 3-hour team experience |
| The Immersion Retreat | 04-immersion-retreat.md | Individual page for the full-day retreat |
| Science of Music & Wellbeing | 06-science-of-music-wellbeing.md | Optional — builds credibility with corporate buyers |

---

## Navigation

Add a **"Workplace Wellbeing"** item to the main menu that links to the overview page.

Consider a dropdown or sub-menu:
- Workplace Wellbeing (overview)
  - The Reset Experience
  - The Connection Experience
  - The Immersion Retreat
  - The Science (optional)

---

## Keeping the Look and Feel

1. **Use the same template/layout** as existing pages (header, footer, font choices, colours)
2. **Match the visual style** — if the site uses a particular hero style, image treatment or section layout, replicate it
3. **Consistent CTA buttons** — use the same button style as the rest of the site
4. **Match tone in imagery** — even with stock photos, aim for the same mood (warm, professional, inclusive)

---

## Step-by-Step: Adding the Main Page

1. In Wix Editor: **Pages panel** (left) → **Add Page**
2. Choose **Page** (standard blank page)
3. Name it: **Workplace Wellbeing**
4. Copy content from `01-workplace-wellbeing-overview.md`:
   - Add a **Hero** section with the headline and subheadline
   - Add **Text** sections for the introduction
   - Add **Gallery** or **Repeater** for the three experience cards (or use three columns with image + text + button each)
   - Add bullet list for "Why Organisations Work with Sounds Joyful"
   - Add the Social Impact section
   - Add the CTA section with contact details
5. **Link the "Learn more" buttons** to the three individual pages (create these first, or add placeholder links)

---

## Step-by-Step: Adding Individual Session Pages

Repeat for each session:

1. **Add Page** → Name it (e.g. "The Reset Experience")
2. Use the same layout structure as other product/service pages on the site if they exist
3. Copy content from the relevant markdown file
4. Add a **table** for "Practical Details" if Wix supports it, or use a simple list
5. Add CTA at the bottom

---

## SEO Settings (for each page)

In **Page Settings** → **SEO**:

- **Title tag:** Use the "Page title (SEO)" from each file
- **Meta description:** Use the "Meta description" from each file  
- **URL slug:** Use the "URL slug" from each file

---

## Content Tips

- **Don't list prices** — use "Sessions are tailored to your organisation. Please contact us to discuss."
- **Use atmosphere images** — people singing together, corporate collaboration, relaxation, connection
- **The HR pitch paragraph** — consider using it in the hero or as a prominent quote/callout on the main page
- **"Why Sounds Joyful"** — use the full version on the main page, short version on session pages if space is tight
